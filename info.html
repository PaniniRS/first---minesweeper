How to play?

1. Launch the Game

To start the game, click on the "Start" button.

2.Uncover Squares

Click on any square on the grid to begin the game. The square will uncover, revealing either a number or a mine. If the square reveals a number, it indicates the number of mines hidden in the adjacent squares. If the square reveals a mine, the game is over.

3.Use Clues

Use the numbers revealed to help you determine where the mines are located. For example, if a square reveals the number 1, there is one mine hidden in one of the adjacent squares. If a square reveals the number 2, there are two mines hidden in the adjacent squares, and so on.

4.Flag Mines

If you think a square contains a mine, right-click on the square to place a flag. This helps you keep track of which squares you think contain mines. If you accidentally flag a square that does not contain a mine, right-click on the square again to remove the flag.

5.Win the Game

The objective of Minesweeper is to uncover all the squares that do not contain mines. Once you have uncovered all the safe squares, the game is won.
